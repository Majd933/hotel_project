# صور الموقع

يرجى إضافة الصورة الرئيسية هنا:

**اسم الملف:** `hero-pool.jpg`

**الوصف:** صورة المسبح الفاخر للفندق (صورة رئيسية)

**المواصفات الموصى بها:**
- الصيغة: JPG أو WebP
- الحجم: 1920x1080 بكسل أو أكبر
- الجودة: عالية (للحصول على أفضل عرض)

**ملاحظة:** بعد إضافة الصورة، ستظهر تلقائياً كصورة خلفية في القسم الرئيسي (Hero Section) من الصفحة الرئيسية.
